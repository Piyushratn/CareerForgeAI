# 🚀 CareerForgeAI – AI-Powered Career Growth Platform

> An AI-driven platform that helps users prepare for interviews, build resumes, and analyze career growth using Generative AI.

---

## 📌 Problem

Job seekers struggle with:
- Lack of structured interview preparation
- Poor resume quality (low ATS score)
- No personalized career guidance

---

## 💡 Solution

CareerForgeAI solves this by:
- Providing AI-powered interview practice
- Generating ATS-optimized resumes & cover letters
- Delivering personalized career roadmaps

---

## 🔥 Features

- 🤖 AI Interview System (1000+ questions)
- 📄 ATS Resume Builder & Cover Letter Generator
- 📊 Real-time Performance Feedback
- 🧠 AI-based Skill Gap Analysis
- 🗺️ Personalized Career Roadmap
- 🌐 Supports 50+ industries

---

## ⚙️ Tech Stack

### Frontend
- Next.js
- React.js
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- PostgreSQL (Neon DB)

### AI / ML
- LangChain
- RAG (Retrieval-Augmented Generation)
- OpenRouter API

---

## 🏗️ System Architecture

User → Frontend (Next.js) → Backend (Node.js) →  
→ LangChain (AI Processing) → Vector DB → Response

---

## 📊 Impact

- 🚀 Supports 50+ industries
- 📈 1000+ interview questions integrated
- ⚡ Improves resume quality (ATS-ready)
- ⏱️ Saves hours of manual preparation
